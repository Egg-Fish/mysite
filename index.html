Bidirectional Type Inference

Category: Type Systems

Language: Haskell


This project implements the bidirectional type inference rules in the paper "Bidirectional Typing" (ACM DL) by Jana Dunfield and Neel Krishnaswami.


This implementation does type inference for the higher-ranked polymorphic lambda calculus, with type annotations.

Provenance Graph Analysis

Category: Cybersecurity

Language: C++, Prolog


This project was part of the Odyssey summer programme (CP2107) in NUS School of Computing, under the supervision of Dr Anis Yusof. The objective was to automatically detect malicious activity using data sources from multiple endpoints.


This project involved parsing system logs from Windows Event Logs, Linux Auditd, and PCAP files, into a unified provenance graph. The analysis portion is powered by a rule-based approach, implemented in Prolog.
